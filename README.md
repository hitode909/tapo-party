# tapo-party

Party Hard Mode for TAPO's multi color LED light.
This script keep changing the colors of your home.
This is useful for home party.

```
$ npm ci
$ TAPO_USERNAME='***@example.com' \
  TAPO_PASSWORD='...' \
  npm start
```
